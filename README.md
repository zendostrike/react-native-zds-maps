# react-native-zds-maps

A Google Maps Native View for React Native

## Installation

```sh
npm install react-native-zds-maps
```

## Usage


```js
import { ZdsMapsView } from "react-native-zds-maps";

// ...

<ZdsMapsView style={{}} markers={[]} onMarkerClick={(event) => {}} />
```
